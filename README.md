# Aasara Pension Face Verification

A smart beneficiary verification system for **Aasara Pension** that uses **face recognition and liveness detection** to help elderly and disabled citizens complete pension verification easily.

## 📌 Project Overview

Traditional pension verification methods like **fingerprint and iris scan** often fail for elderly beneficiaries because of worn fingerprints, eye issues, and physical limitations.

This project provides a **face-based verification system** where the beneficiary enters their **Pension ID**, the system captures a **live face image**, performs **liveness detection**, and verifies the face with the stored registered data.

This makes the process:

* Elderly friendly
* Faster
* Non-intrusive
* More reliable

## 🧠 Working Flow

1. User enters Pension ID
2. System fetches beneficiary details
3. Camera captures live face
4. Liveness check confirms real person
5. Face is compared with registered data
6. Verification result is displayed
